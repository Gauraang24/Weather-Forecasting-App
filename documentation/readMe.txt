# Weather Forecasting App

A web application that provides weather forecasts using the OpenWeatherMap API. The app allows users to search for weather information by city and view detailed weather metrics, including temperature, humidity, wind speed, and more.

## Features

- Search for weather data by city
- Display current weather conditions
- Show detailed weather metrics
- Save recently searched cities
- Responsive design for iPad Mini and iPhone SE
- Loading spinner during data fetch
